# 🩺 Care & Cure System  

A system analysis and design project focused on developing a smart healthcare support system to assist patients in managing their daily health activities.

The project aims to improve accessibility to healthcare services through reminders, consultation, and pharmacy support.

## The work included:
- Defining business and stakeholder requirements  
- Identifying functional and non-functional requirements  
- Designing system models (Use Case and Context diagrams)  
- Planning system architecture and components  
- Analyzing real user needs through interviews  

## 📊 Key Features:
- Medication reminders and health alerts  
- 24/7 medical consultation support  
- Searching for nearby pharmacies and medicine availability  
- Health tracking and reporting  
- Simple and user-friendly interface  

 ## 📌 Outcome:
A complete system design that demonstrates how a digital healthcare assistant can improve patient experience, safety, and accessibility to medical services.
